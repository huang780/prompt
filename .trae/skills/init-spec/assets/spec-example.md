## 2. 功能需求详述

### 2.1 快速记录（F1）

#### F1.1 语音记录
- **前置条件**：用户已授权麦克风权限。
- **流程**：
  1. 用户长按首页“🎙️ 按住说话”按钮。
  2. 开始录音，按钮变为“松手结束”。
  3. 用户松手，调用语音转文字API。
  4. 识别成功后，自动填充内容文本框，弹出半屏编辑弹窗。
  5. 用户补充/修改内容、预估耗时（分钟）、选择标签。
  6. 点击“保存”，小事记录生成，状态为“待处理”。
- **异常处理**：
  - 录音时长<0.5秒 → 提示“说话时间太短，请重试”。
  - 语音转文字超时（>3秒）或失败 → 提示“识别失败，请手动输入”。
  - 未选择标签 → 提示“请至少选择1个标签”，禁止保存。
  - 未填写预估耗时 → 默认填入5分钟，允许用户修改。

#### F1.2 手动输入
- **流程**：
  1. 点击“手动输入”按钮，弹出半屏输入面板。
  2. 用户输入内容文本、耗时、标签。
  3. 可选添加照片/录音。
  4. 保存。
- **约束**：内容文本不能为空，不能超过200字。

#### F1.3 添加证据
- **照片**：最多3张，支持拍照或相册选择。每张图片压缩至宽度750px以内，格式JPEG。
- **录音**：最长30秒，格式AAC。录音前需授权麦克风。
- **存储**：上传至云存储，路径 `{openid}/{recordId}/{type}/{timestamp}`，私有读写。

#### F1.4 标签系统
- **预定义标签**（不可删除，可重排序）：
- **自定义标签**：用户最多可添加5个，长度4~8字，不能与预定义重名。
- **保存时标签选择**：支持单选或多选（最多3个标签）。

### 2.2 小事清单（F2）

#### F2.1 列表展示
- **默认排序**：按创建时间倒序（最新在上）。
- **分组**：今天 / 昨天 / 更早（按日期折叠）。
- **每条卡片信息**：
- 图标（根据标签动态显示）
- 内容缩写（最多20字，超出…）
- 耗时（分钟）
- 标签（最多显示2个）
- 状态圆点（🟡待处理 / ✅已处理）
- 证据标记（📷/🎙️ 如有）

#### F2.2 筛选
- **状态筛选**：全部 / 待处理 / 已处理 / 证据（只要有照片或录音）。
- **标签筛选**：下拉多选标签，支持清空。
- **日期筛选**：快捷选项（今天、昨天、本周、本月） + 自定义起止日期。
- **含证据筛选**：仅显示有照片或录音的记录。

#### F2.3 操作
- **左滑**：显示“标记完成”和“删除”按钮。
- 标记完成：状态变为“已处理”，不可再标记完成，可改为“未完成”。
- 删除：软删除，记录移至回收站（保留30天可恢复，前端不展示恢复入口，后台可做）。
- **长按**：进入多选模式，可批量标记完成、批量删除、批量导出（按当前筛选结果）。
- **点击卡片**：跳转至详情页（F3）。

### 2.3 小事详情（F3）

#### F3.1 查看与编辑模式
- 默认查看模式，点击“编辑”按钮进入编辑模式。
- 可修改字段：内容文本、耗时、标签、处理状态、处理备注。
- 可添加/删除照片、重新录音（覆盖原录音）。

#### F3.2 追加证据
- **添加照片**：点击“+”选择，最多3张，超过时提示“已达上限”。
- **重新录音**：点击已有录音的“重新录制”按钮，新录音覆盖旧录音。

#### F3.3 转发与导出
- **转发给微信好友/群**：生成小程序卡片，卡片内容包含：
- 标题：小事清 · {标签名}
- 描述：内容前20字 + “耗时X分钟”
- 图片：第一张照片或默认图标
- 点击后进入该小事详情页（需用户已关联openid，否则先登录）
- **生成证据截图**：生成长图（宽度750px），包含所有文字、照片缩略图、时间戳，保存至相册。

### 2.4 时间小偷分析器（F4）

#### F4.1 数据范围
- **默认**：本周（周一00:00:00 至 周日23:59:59）
- **切换选项**：上周、本月、上月、自定义区间（最多跨度90天）。

#### F4.2 统计维度
- **总被偷时间**：选中范围内所有小事耗时之和（分钟）。
- **饼图**：按标签汇总耗时，占比小于2%的标签并入“其他”。
- **偷时排行榜**：按标签汇总耗时降序，展示前5名，其余折叠。

#### F4.3 钻取
- 点击饼图扇区或排行榜项 → 展示该标签下所有小事列表（可点击查看详情）。
- 列表支持导出（生成本标签的小事报告）。

#### F4.4 维权证据包
- 自动筛选范围：当前统计区间内，标签为 #加活、#甩锅、#加班、#顾客无理 的小事。
- 生成内容：每条小事包含时间、内容、耗时、标签、照片缩略图（如有）、录音转文字（如有）。
- 导出格式：PDF（A4大小，适合打印）或长图（PNG）。
- 导出前可预览，取消某些不想要的条目。

#### F4.5 分享周报
- 生成卡片：饼图截图 + 总耗时 + TOP3偷时项。
- 卡片底部带小程序码，扫码后可进入分析器页面。

### 2.5 证据箱（F5）

#### F5.1 列表展示
- 只展示包含至少一张照片或一条录音的小事。
- 卡片样式同小事清单，但额外显示“证据数量”（图片张数+录音标记）。

#### F5.2 筛选与操作
- 支持按标签、日期范围、内容关键字筛选。
- 支持多选，批量“添加到导出”。
- 导出功能与F4.4共用PDF生成能力。

### 2.6 微信生态集成（F6）

#### F6.1 服务通知
- **订阅**：用户首次进入“我的”页面时，弹出订阅授权对话框（仅一次），引导开启。
- **发送规则**：每天用户设定的时间（默认18:00）发送一条模板消息。
- **模板内容**：
```json
{
  "thing1": {"value": "今日共记录了 {{count}} 件小事"},
  "time2": {"value": "{{totalTime}} 分钟"},
  "thing3": {"value": "请点击前往清理未完成项"}
}
