# 文档结构与提纲规范 (Structure Blueprint)

## 3.1 元结构原则
所有文档遵循"倒金字塔+模块化"原则：
- 顶部20%：核心结论/主旨全景（让读者30秒内抓住重点）
- 中部60%：分层论证/展开（每部分可独立阅读）
- 底部20%：总结升华/行动号召（回扣开头，形成闭环）

## 3.2 通用章节骨架
```text
[标题区]
  └─ 标题：黑暗中的诗意与光明——读《假如给我三天光明》与《飞鸟集新月集》有感

[导语区]（占全文5-10%，约150-300字）
  └─ 三要素：背景（为何读这两本书）+ 核心发现（精神共鸣）+ 结构预告（怎么读）

[主体论证区]（占全文60-70%，约1800-2100字）
  └─ 采用"MECE原则"：相互独立，完全穷尽
  └─ 每章遵循"论点→论据→论证→小结"四步循环

[结论/行动区]（占全文10-15%，约300-450字）
  └─ 不引入新论据，仅整合前文并指向未来
```

## 3.3 读书心得定制模板

```text
1. 导语（约200字）
   - 阅读背景：为何选择这两本书
   - 核心发现：海伦·凯勒与泰戈尔的精神共鸣
   - 结构预告：本文将从四个方面展开

2. 海伦·凯勒与泰戈尔的精神内核（约500字）
   - 海伦·凯勒：在黑暗中寻找光明
   - 泰戈尔：在平凡中发现诗意
   - 两者的共同点：对生命的热爱、对自然的敏感、对精神世界的追求

3. 两本书主人公的相似之处（约600字）
   - 对生命的热爱：海伦对三天光明的珍惜 vs 泰戈尔对日常生活的赞美
   - 对自然的敏感：海伦通过触觉感知世界 vs 泰戈尔通过诗歌描绘自然
   - 对精神世界的追求：海伦的内心强大 vs 泰戈尔的哲学思考

4. 这些精神在大学生活中的具体体现（约600字）
   - 学习态度：从海伦身上学到坚持，从泰戈尔身上学到诗意
   - 人际关系：学会珍惜身边的人，学会用诗意的方式看待他人
   - 自我认知：在困境中保持希望，在平凡中发现美好

5. 精神层面的改变（约600字）
   - 价值观的改变：从功利主义转向对生命本质的追求
   - 人生目标的调整：从单纯追求成绩转向全面发展
   - 内心力量的增强：面对困难时的勇气和韧性

6. 结论（约300字）
   - 总结两本书对我的影响
   - 展望未来：如何将这些精神融入日常生活
   - 给读者的启示
```

## 3.4 段落微观结构
- **起承转合标记**：每段开头使用逻辑路标词（"具体而言"、"反观之"、"值得注意的是"、"然而"、"因此"）
- **信息密度**：每段不超过5个信息点，超过则拆分
- **过渡规则**：段与段之间必须有显性连接词或隐性逻辑链（禁止硬跳转）

## 3.5 长度控制公式
- 总字数 = 基础值 × 复杂度系数
  - 基础值：中篇(3000)
  - 复杂度系数：对比分析(1.3) → 总字数约3900字

## 3.6 特殊结构要求
- **对比分析**：在对比海伦·凯勒与泰戈尔时，使用并列句式增强对比效果
- **个人化叙事**：每提出一个观点，必须有具体的例子或场景支撑
- **层次递进**：从精神内核→相似之处→具体体现→精神改变，层层深入
- **首尾呼应**：结论部分必须回应导语中提出的问题
