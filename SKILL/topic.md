# 主题锚定规范 (Topic Anchoring)

## 1.1 核心主题声明
- **主命题**：海伦·凯勒与泰戈尔的精神共鸣——在黑暗中寻找光明，在平凡中发现诗意，以及这种精神如何重塑一名软件工程专业大学生的世界观与人生观
- **边界范围**：
  - 必须涵盖：
    1. 海伦·凯勒与泰戈尔的精神内核对比分析
    2. 两本书主人公的相似之处（对生命的热爱、对自然的敏感、对精神世界的追求）
    3. 这些精神在大学生活中的具体体现（学习态度、人际关系、自我认知）
    4. 精神层面的改变（价值观、人生目标、内心力量）
  - 明确排除：
    1. 不详细介绍海伦·凯勒的生平年表或泰戈尔的创作历程
    2. 不讨论两本书的文学技巧或语言风格分析
    3. 不进行泛泛而谈的"读书感悟"而无具体事例支撑
- **目标读者**：大学生群体、文学爱好者、对个人成长感兴趣的读者

## 1.2 偏题检测机制
生成过程中，每完成一个章节需自检：
- [ ] 本节内容是否直接支撑主命题？
- [ ] 若删除本节，主命题是否仍然完整？（如答案是"是"，则本节可能冗余）
- [ ] 本节是否引入了边界范围外的新话题？

## 1.3 题材适配示例
| 题材类型 | 主命题示例 | 边界排除示例 |
|---------|-----------|------------|
| 读书心得 | 《假如给我三天光明》与《飞鸟集新月集》中主人公的精神共鸣及其对大学生精神成长的启示 | 不介绍作者生平年表、不讨论该书出版市场反响、不进行纯文学技巧分析 |

当内容偏离时，使用以下句式强制回归：
"回到核心命题：海伦·凯勒与泰戈尔的精神共鸣及其对大学生精神成长的启示，我们需要聚焦于..."
